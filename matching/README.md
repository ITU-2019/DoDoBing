# Requirements

- python 3.x
- pip

# Install

```bash
$ pip install nose
```

# Running tests
_Unit tests_
```bash
$ nosetests -d Python/testUtils.py
```
[Reference to nosetests](http://pythontesting.net/framework/nose/nose-introduction/)

_Running all examples (requires bash)_
```bash
$ Python/scriptExample.sh
```
